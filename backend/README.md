# Requisitos Funcionais – Backend S.O.S. na Mão

## 1. Autenticação e Usuários

- [ ] Cadastro de usuário: email, senha (via Supabase Auth)
- [ ] Login/Logout: via Supabase Auth
- [ ] Recuperação de senha: email de redefinição via Supabase
- [ ] Atualização de perfil: nome, telefone, contato de emergência
- [ ] Validação de autenticação: middleware Express para proteger endpoints
